Pedro Pathing Documentation

Pedro Pathing is a Reactive Vector Follower designed by FTC Team 10158 Scott's Bots to improve autonomous pathing efficiency. By leveraging advanced Bezier curve generation, it achieves smoother, faster, and more optimal paths compared to traditional systems like RoadRunner.

This documentation provides a comprehensive guide to help you set up Pedro Pathing, including localization methods, automatic tuners, PID tuners, and example implementation code for both autonomous and teleop modes.

Whether you're a novice or a seasoned robotics team, this guide ensures you can effectively integrate Pedro Pathing into your robotics projects and fully utilize its capabilities.
